# V26 스킬 점수 계산기

타자/투수 카드의 스킬 조합 점수를 계산하고,
카드 타입별 기준 확률과 판정 등급을 확인하는 웹 계산기입니다.

## 실행 방법

```bash
npm install
npm run dev
```

빌드/검증:

```bash
npm run build
npm run lint
```

---

## 작업 기록

### v0.3.1

- 계산/시뮬 결과 가독성 개선
  - 스킬 3개 각각의 점수를 결과와 함께 표시
  - 결과 패널에 판정등급 기준 설명 추가
- 고급스킬변경권 시뮬 UX 개선
  - 목표 판정등급 도달 시까지 자동 롤 지원
  - 세션 사용 횟수, 최고 점수, 달성 메시지 표시
- 모바일 UI 개선
  - 현재 결과 요약 박스에서 스킬 목록을 가로 스크롤 대신 세로로 표시
  - 버튼/패널 간격과 높이를 줄여 한 화면에서 더 많이 보이도록 조정
  - 스킬 변경 시 레이아웃 폭이 흔들리지 않도록 스크롤/폭 계산 안정화
- 입력 방식 개선
  - 카드 종류를 드롭다운 대신 버튼형 선택으로 변경
  - 계산 대상 버튼 크기와 여백을 모바일 기준으로 추가 보정
- 데이터 정합성 보정
  - 선발 `마당쇠` 제거
  - `국대에이스(중복)` 표기를 `국대에이스`로 통일
  - 국가대표 전용 스킬 노출 조건 및 투수 스킬 가용성 규칙 보정
  - 점수 차이가 있는 변형 스킬은 시뮬에서 별개 스킬로 유지

### v0.3.0

- 고급스킬변경권 시뮬 기능 추가
  - `스킬 계산기 / 고급스킬변경권 시뮬` 2개 탭 구성
  - 버튼 1회 클릭으로 고급스킬변경권 1회 사용 결과를 바로 확인
  - 타자/선발/중계/마무리별 확률 프로필 반영
- 자동 롤 기능 추가
  - 목표 판정등급(`C / A / S / SSR+`) 선택
  - 목표 등급 이상 달성 시까지 자동 롤
  - 세션 사용 횟수, 최고 점수, 달성 메시지 표시
- 시뮬 결과 정합성 보정
  - `철완`, `좌승사자`, `원투펀치`, `라이징스타`, `도전정신`, `패기` 계열은 중복 등장하지 않도록 가족 단위 처리
  - 골든글러브 기본 레벨은 `6/6/6`, 그 외 카드는 `6/5/5`로 자동 설정
- 결과 체계 정리
  - 판정등급을 `F / C / A / S / SSR+`로 통일
  - 결과 패널에 판정등급 기준 설명 추가
- 모바일 UI 개선
  - 시뮬 버튼 바로 아래에 현재 결과 요약 박스 추가
  - 현재 스킬 3개, 점수, 확률, 등급을 한 화면에서 바로 확인 가능
  - 계산 대상 버튼 정렬/여백 보정
