# V26 스킬 점수 계산기

타자/투수 카드의 스킬 조합 점수를 계산하고,
카드 타입별 기준 확률과 판정 등급을 확인하는 웹 계산기입니다.

## 실행 방법

```bash
npm install
npm run dev
```

빌드/검증:

```bash
npm run build
npm run lint
```

---

## 작업 기록

- 보수적 구조 정리
  - 루트 `src/data/skills.ts`의 중복 타자 데이터 원본 제거

### v0.3.3

- 임팩트 스변 시뮬 탭 추가
  - `임팩트 스변 시뮬` 새 탭 구성
  - 임팩트 카드는 `1스킬 고정`으로 처리하고, 사용자가 고정 스킬을 직접 선택 가능
  - 일반 스킬변경권 확률표 기준으로 `2, 3번 슬롯이 둘 다 메이저`가 나올 때까지 자동 롤
  - 성공 시 몇 번 만에 나왔는지, 누적 사용 횟수, 마지막 성공 횟수 표시
- 시뮬 구조 정리
  - 일반 스킬변경권/고급 스킬변경권이 공통 롤 코어를 공유하도록 분리
  - 스킬 가족 묶음 규칙과 스킬군 중복 방지 로직 재사용 구조로 정리

### v0.3.2

### v0.3.0

- 고급스킬변경권 시뮬 기능 추가

- 자동 롤 기능 추가
  - 목표 판정등급(`C / A / S / SSR+`) 선택
  - 목표 등급 이상 달성 시까지 자동 롤
  - 세션 사용 횟수, 최고 점수, 달성 메시지 표시
- 시뮬 결과 정합성 보정
  - `철완`, `좌승사자`, `원투펀치`, `라이징스타`, `도전정신`, `패기` 계열은 중복 등장하지 않도록 가족 단위 처리
  - 골든글러브 기본 레벨은 `6/6/6`, 그 외 카드는 `6/5/5`로 자동 설정
- 결과 체계 정리
  - 판정등급을 `F / C / A / S / SSR+`로 통일
  - 결과 패널에 판정등급 기준 설명 추가
- 모바일 UI 개선
  - 시뮬 버튼 바로 아래에 현재 결과 요약 박스 추가
  - 현재 스킬 3개, 점수, 확률, 등급을 한 화면에서 바로 확인 가능
  - 계산 대상 버튼 정렬/여백 보정
