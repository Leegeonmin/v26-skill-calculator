# V26 스킬 점수 계산기

타자/투수 카드의 스킬 조합 점수를 계산하고,
카드 타입별 기준 확률과 판정 등급을 확인하는 웹 계산기입니다.

## 실행 방법

```bash
npm install
npm run dev
```

빌드/검증:

```bash
npm run build
npm run lint
```
